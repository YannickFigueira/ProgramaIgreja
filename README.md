# Programa Igreja

Aplicativo para exibição de slides e textos da Harpa Cristã em DataShow, desenvolvido em Python.

## 📦 Estrutura do Projeto

- `programaigreja.py` → arquivo principal (ponto de entrada)
- `dados.py` → módulo auxiliar para manipulação de dados
- `slide.py` → módulo auxiliar para exibição de slides
- `BS Para DataShow – PowerPoint/` → recursos de apresentações
- `HarpaTexto/` → textos da Harpa Cristã

## 🚀 Instalação

### Via pacote `.deb`
```bash
sudo dpkg -i programaigreja-0-2-0.deb
sudo apt-get install -f   # corrige dependências se necessário

### Via Pyinstaller`
```bash
pyinstaller programaigreja.spec
